Timestamp Microservice

How to use:

We provide you an API endpoint that you can use to get your desired timestamp.

All you need to do is provide a suitable date by one of the following manners:

  • Provide specific date in YYYY-MM-DD.
  • Provide specific date as full date (01 January 2020).
  • Provide Unix timestamp (in milliseconds).

Example Usage:

  • [url]/api/1996-07-27
  • [url]/api/1629280395000

Expected Result:

{ "unix":"1629280395000", "utc":"Wed, 18 Aug 2021 09:53:15 GMT" }
